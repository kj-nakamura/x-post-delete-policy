X Post Delete Tool - プライバシーポリシー

1. データ収集について

X Post Delete Toolは、ユーザーのいかなる個人情報も収集しません。この拡張機能は、ユーザーのXのポストを削除するための機能を提供しますが、以下のデータを収集することはありません：

  • 個人を特定できる情報
  • 健康に関する情報
  • 財務状況や支払いに関する情報
  • 認証に関する情報
  • 個人的コミュニケーション
  • 位置情報
  • ウェブ履歴
  • ユーザーのアクティビティ
  • ウェブサイトのコンテンツ

2. データの保存と転送

この拡張機能はユーザーデータを収集しないため、外部サーバーへのデータ転送や保存は一切行いません。すべての操作はユーザーのブラウザ内でローカルに実行されます。

3. 権限の使用

この拡張機能は以下の権限を使用します：

  • scripting: X（Twitter）のウェブページ上でポスト削除スクリプトを実行するために使用します。
  • activeTab: ユーザーが現在表示しているX（Twitter）のタブでのみ動作するために使用します。
  • ホスト権限: X（Twitter）のウェブサイト（twitter.comおよびx.com）でのみ動作するために使用します。

4. サードパーティサービス

この拡張機能はサードパーティのサービスやAPIを使用しません。

5. 変更

このプライバシーポリシーは将来変更される可能性があります。変更があった場合は、拡張機能のページで通知します。
